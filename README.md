## ✨ Features

- **🚀 Ready to Use:** Drop-in HTML and CSS for instant beautiful UI.
- **📱 Fully Responsive:** Grid and Flexbox layouts that adapt perfectly to any device size.
- **🎨 Modern Design:** Follows the latest web design trends (glassmorphism, subtle gradients, soft shadows).
- **♿ Accessible:** Semantic HTML markup for screen readers and keyboard navigation.
- **⚡ Lightweight:** Pure CSS with zero heavy JavaScript dependencies (unless necessary for interactions).

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/venkatesh0963/ui-craft.git
   ```

2. **Open the Project:**
   Simply open `index.html` in your favorite browser to view the component dashboard.

3. **Copy the Code:**
   Navigate to the component category you need, view the video preview, and hit **View Code** to copy the raw HTML and CSS into your own project!
